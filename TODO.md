# Рекомендации по проекту

- README.md сделайте сборку проекта понятной для новых людей.
- Рассмотреть возможность подключения Prittier.
- Настроить app.vue правильно.
- Сделать default.vue layout.
- Раскидать компоненты по папкам в том числе из ./app/layouts .
- Рассмотреть ssr: false.
- Рассмотреть position: sticky.
- Создать ./.env.example .
